How to execute the source code..

1) On any Unix system cd to the folder xv6.
2) In the terminal type 'make'.
3) After the source code is compiled type 'make qemu' which will boot xv6 in a new QEMU window.
4) Type 'ps' in the xv6 shell to execute the ps program which will display active process and with different parameters

Your system must have gcc compiler and QEMU to compile and boot xv6. You can download it on your
system using the following commands:

sudo apt-get update
sudo apt-get install build-essential
sudo apt-get install gcc-multilib
sudo apt-get install qemu
